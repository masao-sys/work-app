# WORK APP
職場内の報告アプリを作成しました
- 職場内の大切な連絡事項を社員へシェア
- 社員が連絡事項を確認したのかが分かる
- 部署ごとの掲示板にてタスク管理

下記のページにて公開中です<br>
https://work-pf.herokuapp.com/
